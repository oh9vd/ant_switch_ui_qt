# Mast unit

Remote unit that is hard wired with 2x6 relay module. This unit has LoRa radio listening commands from Bridge unit and sending responses.

## Arduino IDE build

- **Board settings**: [Wisblock RAK11300](https://raw.githubusercontent.com/RAKwireless/RAKwireless-Arduino-BSP-Index/main/package_rakwireless_index.json)
